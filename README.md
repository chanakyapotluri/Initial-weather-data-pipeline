# Weather Data Pipeline

## Overview
This project fetches real-time weather data from an API, processes it using Python, and stores it in a SQLite database.

## Features
- Fetches live weather data using API
- Stores data in SQLite database
- Retrieves latest records using SQL queries
- Handles errors gracefully

## Tech Stack
- Python
- Requests (API calls)
- SQLite
- SQL

## How to Run
1. Install dependencies:
   pip install requests

2. Run the script:
   python main.py

## Output
- Weather data stored in `weather.db`
- Latest records printed in terminal
